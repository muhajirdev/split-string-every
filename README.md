## Installation

```bash
npm install --save split-string-every
```

## Usage

```javascript
import { splitStringEvery } from "split-string-every";
```

or

```javascript
import splitStringEvery from "split-string-every";
```

```javascript
splitStringEvery("c310d92e1efc6521beeebc80336fc98b", 8, " "); // "c310d92e 1efc6521 beeebc80 336fc98b"
```
